# ZeroTruster

Personal website for IT Security & Compliance — [www.zerotruster.com](https://www.zerotruster.com)

## Stack

- Static single-page site (HTML / CSS / JS)
- PWA-enabled with service worker & offline support
- Particle animation background with mouse interaction
- Fully responsive (mobile-first breakpoints)

## Local Development

Open `index.html` in any browser — no build step required.
